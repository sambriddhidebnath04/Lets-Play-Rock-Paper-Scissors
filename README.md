# 🎮 Rock Paper Scissors Game  

A simple and interactive web-based Rock Paper Scissors game where you can play against the computer.  

Built to practice core JavaScript logic, handling user interactions, and creating a clean, responsive interface.  

---

## 🐾 Features  

• Play Rock, Paper, Scissors against the computer  
• Instant result display (win/lose/draw)  
• Dynamic score tracking  
• Clean and minimal user interface  
• Fully responsive for desktop and mobile  

---

## 🛠️ Tech Stack  

• HTML  
• CSS  
• Vanilla JavaScript  
• Vercel (for deployment)
